INTRODUCTION:

Data Quality Analysis is an integral step in any data analytics based project. Real world data is not always well behaved. Many decisions need to be made in a consistent manner before diving into the analysis step. The main issues in this dataset is the presence of null values or empty fields. Another common problem which is dealt here is the presence of outliers which can be seen in the upcoming sections. The ability to recognize the existence of such conditions helps to choose appropriate statistics and data cleaning steps used in analysis. It also drives the flow of logic in creating charts and groups as per the decision and property being analyzed.

In the following sections select fields can be seen with the mentioned properties. Also the final sections includes a comprehensive list of all the the fields in the datasets and their associated quality analysis.
