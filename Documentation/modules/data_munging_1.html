INTRODUCTION:

Data Munging is an important step in acquiring a structure and control on the datasets. In this analysis, the objective is to be able to focus on a certain type of market at any point of time and to be able to switch to another one as per the requirement. Hence, in the section discussing the filtering aspect, the generic approach implemented is discussed which achieves the mentioned functionality. The housing price values are discussed and it can be seen that they are not consistent. Also, the presence of less commonality in between zipcodes of both datasets is a feature worthy of mention. Associated with that discussion is the knowledge about common zipcodes which can be set as a start point for the analysis to begin. However, the decisions must not be biased upon those values only and other options also must be considered which can be seen in Data Analysis and Visualization discussions.
